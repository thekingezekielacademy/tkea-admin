# Standalone Live Classes - Student Guide

## Overview

Standalone Live Classes are special live classes that exist independently from regular courses. They appear only in the Live Classes section and are perfect for workshops, special sessions, or one-time events.

## Features

### 1. **Independent from Courses**
- Standalone live classes are not part of any course
- They have their own title, description, and cover image
- They appear only in the Live Classes section

### 2. **Public Access (First 2 Classes)**
- The first two classes of any standalone live class are **FREE** and **publicly accessible**
- No sign-in or sign-up required to watch the first 2 classes
- Anyone can access these classes directly via URL

### 3. **Flexible Video Sources**
- Standalone live classes can use:
  - YouTube videos (any YouTube URL)
  - Direct video URLs (MP4, webm, etc.)
  - Embedded video links

## How to Access Standalone Live Classes

### Option 1: Direct URL (Public Access)
For the first 2 free classes, you can access them directly:
```
https://your-app.com/live-classes/session/{sessionId}
```
No authentication required!

### Option 2: Via Live Classes Section
1. Navigate to the **Live Classes** section in the app
2. Browse available standalone live classes
3. Click on a class to join the scheduled session

### Option 3: From Course List
- Standalone live classes do NOT appear in the courses list
- They are only visible in the Live Classes section

## Viewing a Live Class Session

### What You'll See

1. **Class Header**
   - Class title (for standalone) or Course title (for course-based)
   - Session type (Morning/Afternoon/Evening)
   - Scheduled date and time
   - Free class badge (if applicable)

2. **Video Player**
   - YouTube videos: Played via YouTube embed
   - Direct videos: Played via HTML5 video player
   - Supports all standard video controls

3. **Class Information**
   - Session description
   - Video title and description
   - Remaining slots (for paid classes)

|---------|-----------|--------------|
| **Source** | Independent | Part of a course |
| **Title** | Custom title | Course title |

| **Cover Image** | Custom cover image | Course cover image |

## Troubleshooting

### Issue: "Untitled" Showing
If you see "Untitled" for a standalone live class:
- This means the class title was not set by the admin
- Contact support to have the title updated
- The class will still function normally

### Issue: Video Not Playing
- Check your internet connection
- Try refreshing the page
- Verify the video URL is valid
- Contact support if the issue persists

### Issue: Can't Access Class 3+
- Verify you're signed in
- Check if payment is required
- Ensure you have available slots
- Contact support if you believe you should have access

## Best Practices

1. **Bookmark Free Classes**: Save URLs for the first 2 free classes for easy access
2. **Check Schedule**: Review the schedule before joining to avoid missing sessions
3. **Share Responsibly**: Share public class URLs with others who might benefit
4. **Join Early**: Join sessions early to secure your spot (for paid classes)
